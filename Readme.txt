Display your recent Snipplr Snippets in your sidebar.

== Description ==

Use this widget to display your (recent) Snipplr snippets in your sidebar(s).

You just have to enter your Snipplr API key.

= Features: =

* Define your Snipplr API key, how many Snippets you want to be displayed and in which order they should appear. Choose between date, title and random.
* Choose if you want to exclude private and / or favorite Snippets.
* Use as much Snipplr Widgets as you want. Every Widget has its own settings.
* Data is cached locally to reduce API calls and load time. You can set the cache duration in the Widgets options.

= Languages =
Currently supported languages:

* English (default)
* German (since 1.1.2)

If you created your own translation, feel free to send me your files, so I can include it into the download for everyone.

== Installation ==

 = Use the automatic installer (recommended) or =
1. Unzip the package, and upload `snipplrwidget` to your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
 = After the Plugin is activated, simply add the 'Snipplr' Widget to your sidebar and configure it. =
